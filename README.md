## 📖 About The Project

**College Budgeting System** is a full-stack web application built to help college students take control of their personal finances. Managing tuition, daily expenses, and savings on a student budget is genuinely challenging — this project exists to make that process cleaner, more visual, and actionable.

The system allows students to track income and expenses, categorize spending, and gain insight into their financial habits — all within a clean, modern web interface.

> *Built as a real-world project to apply full-stack development skills with TypeScript, Python, and containerized deployment using Docker.*

---

## ✨ Features

- 📊 **Dashboard Overview** — At-a-glance summary of income, expenses, and balance
- ➕ **Transaction Management** — Add, edit, and delete income or expense entries
- 🏷️ **Category Tagging** — Organize spending by categories (food, transport, tuition, etc.)
- 📅 **Monthly Breakdown** — Track financial trends over time
- 🔐 **User Authentication** — Secure login and personal data management
- 🐳 **Dockerized Backend** — Consistent and portable environment via Docker
- 📱 **Responsive UI** — Works seamlessly across desktop and mobile

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) v18+
- [Python](https://www.python.org/) 3.10+
- [Docker](https://www.docker.com/) (optional, for containerized backend)
- [Git](https://git-scm.com/)

### Running Locally

**1. Clone the repository**

```bash
git clone https://github.com/JihanIsfalana-oss/college_budgeting_system.git
cd college_budgeting_system
```

**2. Set up the backend**

```bash
cd backend_core
pip install -r requirements.txt
python main.py
```

Or run with Docker:

```bash
docker build -t cbs-backend .
docker run -p 8000:8000 cbs-backend
```

**3. Set up the frontend**

```bash
cd web_frontend/college_budgeting_system
npm install
npm run dev
```

**4. Open in your browser**

```
http://localhost:3000
```
